国庆节回家，闲来无事，发现桌子上有一本《管理就是带团队》，于是决定在家的这几天把这本书看下来，并做一个记录。本文章摘要书本中的相关知识，对自己有所帮助。对于管理相关的知识，我们也可以进行探讨，大家可以点击下方进行评论！

# 书本摘要

## 管理哲学：一个人是工作，一群人是事业

1. “三个臭皮匠赛过诸葛亮”，团队协作永远大于孤军奋战
```
（1）优秀的团队，强调的是优势互补、相互协作
（2）让每个员工都融入团队中，彼此相互配合
（3）跨越“部门墙”，让企业各部门之间实现大团结
```
2. 团队成员不仅仅是你的下属，更是你的事业资本
```
purpose partners概念：把员工称为“同志”，即志同道合的伙伴
（1）激情万丈地向员工描述企业愿景
（2）对员工在工作中的付出表示认可
（3）尽可能多让员工参与到决策中来
```
3. 团队精神是企业最核心的竞争力
```
（1）营造“企业兴旺我光荣，企业衰亡我可耻”的文化氛围
（2）把“家人意识”融入团队建设中来
```
4. 凝聚力是团队的保障，创新力是团队的希望
```
（1）在实践与体验中寻找创新灵感
（2）以消费者的身份造访竞争对手
```
5. 好团队的标准：谁都可以用得上，谁都可以离得开
```
团队里的人谁都可以离开，包括管理者，这才不失为一个好的团队
（1）努力培养员工独立解决问题的能力
（2）不再过细地监督下属的工作，而要用好的结果要求下属
```
6. 打造协作性团队，实现每个人的最大利益
```
应该考虑的因素
（1）能力因素
（2）知识因素
（3）气质因素（气质指的是脾气、性格、秉性等）
（4）年龄因素
```
7. 形成利益和荣誉的共同体，才能形成牢固的团队
```
（1）明确地制定利益分配的方案
（2）在荣誉激励中加入感情的“戏份”
```

## 第一招 自我管理：用人格感召团队，用魅力影响团队

1. 不断丰富自己的知识结构，做学者型管理者
```
非学无以广才
管理者有三大能力：技术能力、人际能力和分析能力，支撑这些能力的事学习能力
（1）向书本学习
（2）向实践学习
（3）向优秀者学习
```
2. 做情绪的主人，管理者应有情绪掌控力
```
（1）给自己敲警钟，提醒不良情绪的危害
（2）运用自我观察法控制不良情绪
    ①深呼吸，让自己集中注意力
    ②观察不良情绪状态下你是想逃避还是想对抗
```
3. 修炼非权力性领导力：让员工自觉追随你
```
（1）自信、谦虚、胸怀宽广
（2）吃苦耐劳的低调精神
（3）平易近人的待人态度
```
4. 自我要求：用放大镜管人，用望远镜管事
```
（1）把“用放大镜管人”和“用望远镜管事”作为对自己的要求
（2）严格要求团队成员保持注重细节的工作态度
```
5. 不要和下属争功劳，争了功劳失了心
```
（1）功劳确实是下属的，毫无疑问要给下属
（2）即便是你的功劳，也别忘了与下属分享
（3）下属尽力了，但成绩不突出，依然要记上一功
```
6. 无关紧要的事情，妥协一下更能彰显你的气度
```
（1）有些事可以先缓一缓，不急于一时解决
（2）有些事无须挑明，装一装糊涂给员工留面子
```
7. 必要时做员工的挡箭牌，保护员工得人心
```
（1）管理者决策失误，导致下属工作出差错时，应主动揽过
（2）管理者督察不力，导致下属出现差错时，应主动揽过
（3）管理者用人不当，导致下属工作出错，应主动揽过
```

## 第二招 制度管理：制度才是真正的老板

1. 严格的制度是企业健康成长的保险栓
```
（1）完善
（2）公平
（3）服众
```
2. 按制度办事，没有规矩不成方圆
```
（1）制定具体的可操作的企业管理制度
（2）制度不仅有规定，还应有处置的方式
```
3. 合理的制度设置应该兼顾公平和效率
```
（1）制度的制定初衷要考虑清楚
（2）制度的设置应该兼顾公平和效率
（3）一个制度的出台应该以保护积极实干者为出发点
```
4. 制度就是原则，制度面前人人平等
```
（1）管理者应该积极做表率
（2）按制度办，树立制度的威信
```
5. 执行力是制度权威存活的生命之源
```
（1）处罚不是目的，而是手段
（2）有罚也要有奖，奖罚分明
```
6. 你不讲制度，大家就会跟你讲条件
```
（1）没有制度的公司，一定要赶快建立和完善制度
（2）有制度一定要严格执行制度，而不是法外开恩
```
7. 杜绝人情，它是制度的腐蚀剂
```
（1）区别人情化与人性化
（2）杜绝人情，但要讲人性
```
8. 杜绝“下不为例”的借口，制度面前没有任何理由
```
（1）制度一旦制定，就要严格执行
（2）按制度办事别忘了教育员工
```

##  第三招 授权管理：管的少，团队才能管得好

1. 合理授权才能凝聚起团队的力量
```
（1）授权要适度，别让下属压力太大，以免影响执行效率
（2）授权之后切忌胡乱干涉，以免打击下属的自信心和积极性
```
2. 授权管理的原则：分散权力，总揽大局
```
（1）逐级授权原则
（2）权责明确原则
（3）抓总线、放支线原则
```
3. 管理者不做太多决策，只做重大决策
```
（1）常问自己：这件事我有必要做吗？
（2）明确自己的角色，有所为，有所不为
（3）大事与小事，其实是相对而言的
```
4. 授权的前提是找到合适的人选
```
（1）看下属是否具有完成某项工作的特长和能力
（2）充分考虑授权对象的个性、脾气、气质等
```
5. 有效授权和盲目放权不是“同门师兄”
```
怎样过问和监督才能避免给下属造成不信任感
（1）讲明工作要求及完成期限
（2）适当关注，而不是质疑
```
6. 最有效的授权是让员工各尽其责
```
（1）人才适合干什么，就让他干什么
（2）划分职权，分配工作，让员工清楚自己的职责
```
7. 缺乏信任的授权不可能达到授权的目的
```
（1）在下属犯错之后，依然信任他、授权给他
（2）授权给那些不太出众的下属，激起他们的信心
```

##  第四招 绩效管理：处罚不是目的，带出效率是宗旨

1. 绩效管理为企业发展保驾护航
```
（1）制订绩效管理计划
（2）收集评估对象的信息
（3）根据信息进行绩效评估
```
2. 合理的绩效考核能促使员工努力拼搏
```
（1）保证考核指标合理、考核人员公正
（2）保证驾校考核的连贯性和持续性
（3）不要忘了奖励的激励作用
```
3. 绩效考核，先要注重角色定位
```
（1）总经理的角色————支持者
（2）人力资源部的角色————组织者
    ①制定绩效管理方案
    ②组织绩效考核的培训沟通
    ③组织绩效的考核工作
    ④对考核结果做出处理
（3）其他各部门负责人的角色————实施者
    ①制定绩效标准
    ②记录员工的绩效
    ③考核员工的绩效
    ④辅导员工改进绩效
（4）广大员工的角色————参与者
```
4. 不断改进公司的绩效评估体系
```
（1）评估体系过于复杂，应努力让绩效评估变得简单
（2）一把尺子衡量所有人，应让评估标准保持灵活性
```
5. 与员工面对面地进行绩效评估
```
（1）面谈之前做好准备工作
（2）与不同类型的员工面谈
（3）把握好绩效面谈的原则
    ①彼此信任
    ②鼓励员工说话
    ③不要评价员工的性格
    ④谈论过去和未来
```
6. 多劳多得，绩效与薪资直接挂钩
```
（1）认清绩效与薪资直接挂钩的缺点
    ①可能引发员工之间的不良竞争，员工之间不愿意分享经验，甚至争夺客户
    ②员工绩效同组织利益不一致，员工绩效提高了，组织的绩效反而降低
    ③员工为了追求高绩效而损害客户的利益
（2）强调团队绩效，将团队绩效纳入考核范围
```
7. 额外设置特别薪酬，让员工大胆PK
```
（1）奖金计划应该保持简单
（2）额外奖金制度不要在中途改变规则
（3）额外奖金要及时发放
```

##  第五招 流程管理：没有规范流程，管理一切为零

1. 简单、明确和统一的目标是成功的保证
```
（1）目标必须清晰、明确
（2）目标的制定应切合实际
（3）团队内部达成目标共识
```
2. 用流程规范大家做什么、怎么做
```
（1）既要在流程中明确各员工的职责，还应衔接好不同工作之间的关系
（2）鼓励员工在流程规范工作的基础上主动、灵活地工作
```
3. 落实岗位责任制，谁的工作谁负责
```
（1）通过奖优罚劣的举措推行岗位责任制
（2）谁的工作谁负责，谁出问题谁承担
```
4. 撤销流程中没必要的环节，减少成本浪费
```
（1）流程中某些重复、冲突的环节必须撤销
（2）流程中某几个环节可以由一个人做，就没必要让多人做
```
5. 建立沟通渠道，保证各环节顺利衔接
```
（1）做上传下达的信息通报员
（2）引导员工之间保持高效沟通
```
6. 监管融合，确保工作顺利持久开展
```
（1）重视过问和监督，确保工作情况在自己的掌控之中
（2）监督并不代表不信任，要让员工认识到监督是不可缺的流程
（3）监督不能有一段没一段，必须常态化、习惯化
```

##  第六招 沟通管理：交流不到位，执行要走样

1. 沟通带来理解，理解带来合作
```
（1）管理者应该积极和部属沟通
（2）明确与部属沟通时的几个问题
    ①知道该说什么，即明确沟通的目的
    ②知道什么时候说，即掌握沟通的时机
    ③知道和谁沟通，即明确沟通的对象
    ④知道怎么沟通，即掌握沟通、表达的方式方法
（3）沟通是双向的，引导部属积极沟通
```
2. 重视员工会议，坚决不高“一言堂”
```
（1）别试图当英雄，多给员工发表意见的机会
（2）别仓促做决定，多考虑几种解决问题的方法
```
3. 推心置腹才能解开沟通中的死结
```
（1）以己之心，换人之心
（2）循循善诱，心平气和地引导下属
```
4. 积极接纳意见建议，营造开放式的团队氛围
```
（1）建立沟通渠道，采纳员工意见
（2）创造开放式的办公环境
（3）营造尊重与沟通并存的沟通氛围
```
5. 及时疏导情绪，消除员工心中的负能量
```
（1）不要忽视员工的抱怨
（2）消除员工的浮躁、冷漠、猜疑等负面情绪
```
6. 帮助员工摆脱“无助感”的束缚
```
（1）了解员工表达无助感的几种常见行为（无精打采、行为与往常不太一样）
（2）时常开展一些娱乐活动让大家放松身心
（3）指导和激励你的员工
```
7. 主动沟通，把误解和矛盾消除在萌芽中
```
（1）尽量与员工直接沟通
（2）让员工及时做出反馈
（3）鼓励员工提出疑问
```

##  第七招 激励管理：赶着员工跑是技术，让员工主动跑是艺术

1. 奖励给有贡献的人，福利给有态度的人
```
（1）应该奖励长期行为，而不是短期行为
（2）奖励员工不可忽视有效的沟通
（3）奖励员工不可偏离团队目标
```
2. 物质激励是最有效最直接的激励手段
```
（1）物质激励首先表现为提高员工的薪酬
（2）企业对员工的物质激励要提现公平公正
```
3. 及时奖励员工才能获得最佳效果
```
（1）及时奖励不局限于物质奖励，还包括精神奖励
（2）及时奖励要求奖励快速，但这种快速也不是绝对的
```
4. 真诚的赞美能激发团队成员的热情
```
（1）赞美不可一成不变，而要因人而异
（2）赞美不可虚伪敷衍，而要流露真情实感
（3）赞美切忌不分时机，定要合乎时宜
```
5. 竞争是给员工“注入”的最好的“兴奋剂”
```
流水不腐，户枢不蠹
（1）给员工设置竞争对手，鞭策员工积极进取
（2）给员工设置超越的目标，激励员工的好胜心
（3）引进“鲶鱼”，让团队成员紧张起来
    鲶鱼效应：为了让沙丁鱼活着被打捞、带上海岸，在鱼舱里放入一些鲶鱼。由于鲶鱼是沙丁鱼的天敌，沙丁鱼不得不游动起来，这样就避免了缺氧造成死亡。
```
6. 给予及时反馈也是一种激励
```
（1）定期对员工的近期表现予以点评和指导
（2）设置发言平台鼓励员工发表意见和建议
```
7. 适当的惩罚也是一种必不可少的激励
```
（1）在制度中设定相应的处罚条例，按照制度来惩罚员工
（2）如果提醒和威胁就可以奏效，惩罚就不必要真的实行
（3）惩罚员工不是目的，打一“巴掌”之后“甜枣”要跟上
```

##  第八招 员工管理：没有带不好的队伍，只有不会带队的领导

1. 让马儿奔跑，让鸟儿唱歌，发挥员工的优势
```
（1）不怕有缺点，就怕没特长
（2）合理搭配，使人才优势互补
```
2. 对待天生领袖型员工：大胆授权，带活团队
```
（1）了解天生领袖型员工的特质
    ①思维由远及近，有长远眼光
    ②有使命感，有责任心
    ③善于自我激励，不抱怨不懈怠
（2）对待领袖型员工要大胆授权，舍得下本
```
3. 对待大胆创新型员工：取长补短，择优而用
```
（1）了解创新人才的典型特质
    ①勇于探索的创新精神
    ②敏锐的洞察力
    ③灵活的创造性思维
    ④丰富的创新知识
（2）给创新型员工创造好的工作环境
（3）给创新型员工安排有意义的工作
```
4. 对待埋头苦干型员工：给予指导，高效工作
```
（1）让员工对工作目标有一个清晰的理解
    ①这件事做到什么程度才算成功
    ②哪些方面必须做
    ③哪些方面不用做
（2）引导员工明确一项工作的关键点
    ①对于相似的工作，要教员工反思
    ②对于以前没做过的、全新的工作，要启发员工大致推演一下做好这件事可能要几个关键环节，这对提升工作效率是很重要的
（3）教员工抓好开头，鼓励员工坚持到底
    ①保持平常心去应对
    ②鼓励员工坚持到底
    ③鼓励员工善于利用团队去工作
```
5. 对待恃才傲物型员工：泼泼凉水，精心调教
```
（1）多用感情激励，晓之以理、动之以情
（2）泼冷水，让恃才傲物的员工意识到“公司没他一样玩得转”
```
6. 对待推卸责任型员工：坚决处罚，扫除恶风
```
（1）把好第一道关，别让没责任心的人进门
（2）明确工作任务要求和员工岗位责任
（3）正视员工的推诿行为，用说服与处罚并存的方式处理
```
7. 对待得过且过型员工：加强考核，断其后路
```
（1）健全考核制度，让员工不得不努力
（2）主动接近员工，改变其想法
（3）给他们安排合适的工作
```

##  第九招 结果管理：业绩证明能力，结果说明价值

1. 工作就要以成败论英雄
```
（1）要求员工把工作做到位
（2）鼓励员工追求卓越的效果
```
2. 定好目标精确到个人，别让团队整天“瞎忙”
```
（1）狠抓落实————看下级部门是否真的把团队目标分解到个人
```

##  管理者不可不知的50个团队法则

### 一、团队管理法则

1. 威尔逊法则
```
美国行政管理学家切克·威尔逊认为，领导的指导是员工克服困难的后盾。
当部下得知领导在场负责解决困难时，他们会因此信心倍增。可见，管理者现场指导有助于员工的成长并对团队的成功产生积极的作用。
管理者临场指导可以及时发现并纠正员工的错误，增强员工解决问题的信心，提高员工的工作能力。
```
2. 南风法则
```
南风法则也叫暖风法则，它源于法国作家拉·封丹写的一则寓言：北风和南风比威力，看谁能脱掉行人身上的大衣。北风一发威，寒风刺骨，行人赶紧将大衣裹得紧紧的；南风徐徐
吹动，顿时人间风和日丽，行人全身暖洋洋的，不知不觉就开始解开纽扣，脱掉大衣。在这场较量中，南风取得了胜利。
南风法则告诉我们：对待下属，不能粗暴冷酷，否则，只会激起下属的自卫和逆反情绪。因此，在管理中，要尊重和关心下属，以下属为本，多点人情味，让下属感到温暖，从心
底里打动下属，从而使下属丢掉思想包袱，迸发出工作的激情。
```
3. 同仁法则
```
同仁法则，顾名思义就是把员工当成企业的合伙人，而不是企业的雇员。管理者与员工，实际上是协作关系，大家相互配合，完成公司的工作。
因此，把员工当成合伙人，可以营造良好的人际关系氛围，这样大家的工作效率自然会更高。
```
4. 互惠关系法则
```
给予就会被给予，剥夺就会被剥夺；信任就会被信任，怀疑就会被怀疑；爱就会被爱，恨就会被恨。这就是著名的互惠关系法则。
互惠关系法则说明人与人之间是相互的，对管理者来说，如果你爱自己的员工，员工也会爱你和企业。
```
5. 蓝斯登定律
```
所谓蓝斯登定律，指的是朋友一起工作，比与父亲一起工作有趣得多，它强调的是轻松快乐的工作环境的重要性。
管理者如果重视给员工创造快乐的工作环境，员工就会给公司高效的工作回报。所以，管理者千万别板着脸，表现出一副父亲的严厉模样，不妨放下领导者的架子，与下属交朋友
，这样大家都会快乐，工作效率也会更高。
```
6. 柔性管理法则
```
柔性管理法则指的是以人为中心，对员工进行人格化管理。它的最大特点在于不依靠外力，比如，发号施令、强迫员工做事，而是依靠人性解放、权力平等、民主管理，从内心深
处来激发员工的内在潜力、主动性和创造精神，从而使员工心情舒畅、不余遗力地为企业创造价值。
柔性管理法则的特征是：激励重于控制，务实重于务虚，内在重于外在，心理重于生理，身教重于言教，肯定重于否定。它可以极好地让员工感受到企业的尊重、重视，从而激发
出员工的积极性和创造性。
```
7. 刺猬法则
```
寒冷的冬天，两只困倦的刺猬想拥在仪器取暖，可是由于它们身上都长着刺，因此会彼此刺痛对方，于是，它们不得不隔开一段距离，可是离得太开又无法取暖。几经折腾，它们
终于找到了一个最合适的距离，这个距离也叫安全距离，因为它会让彼此获得对方的温暖，但又不至于被刺伤。这就是著名的刺猬法则。
在团队管理中，管理者也需要牢记刺猬法则，既要与员工成为工作上的朋友，也要与员工保持适度的距离。
```
8. 热炉法则
```
很多人都见过烧热烧红的铁炉，谁要伸手去摸，必然会被烫伤。热炉法则体现了公平感，即不管是谁，只要触碰热炉，都要受到惩罚，没有例外；热炉法则还体现了即时性原则，
即一经触碰热炉，当即受到惩罚，绝不拖泥带水。
其实，企业的规章制度也应该像热炉一样，任何人触碰都会受到惩罚，这是制度权威性的保证，也是约束员工、形成团队纪律的保证。
```
9. 鱼缸效应
```
鱼缸是玻璃做的，透明度很高，从任何一个角度观察，里面的情况都一清二楚。鱼缸法则指的就是在管理中，保持各项管理工作的透明度。
有了透明度，管理者的行为就置于全体下属的监督之下，就可以有效地防止管理者滥用职权，从而强化管理者的自我约束感，增强员工对管理者的信任。
```
10. 酒与污水定律
```
把一匙酒倒进一桶污水，得到的是一桶污水；把一汤匙污水倒进一桶酒，得到的还是一桶污水。这就是“酒与污水定律”。任何团队里，几乎都存在这样的人物，他们存在的目的好
像就是捣乱的，就是为了把事情搞糟。更严重的是，他们就像果箱里的烂苹果，如果不及时处理，会迅速传染，把整箱水果都弄烂。因此，对待“烂苹果”一样的员工，一定要及时
将其扫地出门，决不让他污染团队。
```
11. 破窗效应
```
如果有人打坏了一幢建筑物窗户的玻璃，而这扇破窗没有得到及时修理，人们就可能受到这扇破窗的暗示，继续打烂更多的破窗。时间长了，这栋建筑物就会变得破烂不堪。
破窗效应告诉我们：在公众麻木不仁的气氛中，错误甚至犯罪就会滋生。在企业中，如果管理者对员工的不良行为视而不见，同样会暗示更多的员工表现这些不良行为。比如，员
工随地扔垃圾、不注意节约用水、不及时关灯，久而久之，大家便会将扔垃圾、浪费水电当成一种习惯。所以，管理者要重视最初的错误，及时纠正不良的行为。
```
12. 蝴蝶效应
```
气象学家洛伦兹曾在华盛顿进行了一次名为《一只蝴蝶拍一下翅膀会不会在德克萨斯引起龙卷风》的演讲，这就是著名的“蝴蝶效应”，它说明了1%的错误可能导致100%的失败。
从管理学角度来看，管理者对细节的态度影响工作的质量，尤其是在团队协作中，某一位员工由于忽视了细节，导致工作出差错，有可能导致整个团队的工作出现问题。因为团队
成员的工作相互间有密切的联系，就像生产线上的各个程序，某个程序坏了，其他程序也会无法工作。因此，管理者一定要注重细节，引导员工保持细节精神。
```
13. 海恩法则
```
飞机涡轮展机的发明者德国人帕布斯·海恩提出，每一起严重事故的背后，必然有29次轻微事故和300起未遂先兆以及1000起事故隐患。这就是“海恩法则”。要想消除一次严重的事
故，就必须把这1000起事故隐患控制住。
海恩法则说明：事故的发生是量变到质变的结果；其次，再好的技术、再完美的规章，在实际操作中也无法取代人自身的素质和责任心。因此，在检查环节，决不允许走马观花，
而要用心发现微小的差错。
对企业管理，特别是安全管理来说，海恩法则无疑是一种警示，因为任何一起事故都不是偶然的，都是有征兆的。同时，安全事故也是可以控制的。因此，发现并控制好征兆是管
理者应该重视的工作。
```
14. 马特莱法则
```
马特莱法则又称80:20法则，即二八法则。从管理者角度来看，经营和管理企业时不必面面俱到，而应侧重抓住关键的20%，把主要精力用在关键性的工作上，以保证高效率地工作。
从人力资源管理的角度来看，企业管理者应把主要精力放在20%的业务骨干的管理上，抓住企业发展的骨干力量。再通过这20%的少数优秀员工，去带动80%的多数员工，以提高企业
的工作效率。
从营销的角度来看，企业管理者应抓住占总数20%的重点商品、重点用户，通过专注经营，以形成自己的优势产业，并达到牵一发而动全身的效果。
```

### 二、团队合作法则

15. 木桶定律
```
一只沿口不齐的木桶，盛水的多少，不在于木桶最长的那块木板，而取决于最短的那块木板。要想提高木桶的容量，要做的不是去加长最长的那块木板，而是下工夫补齐最短的
木板。当然，一只木桶的容量还取决于木板间的结合是否紧密，如果木板间存在缝隙，同样无法盛满水，甚至滴水不存。这就是著名的木桶定律。
对一个团队而言，其协作力、竞争力的强弱有时候也不取决于优秀成员有多强，而在于最劣势的那名成员，他会拖团队的后腿，影响团队的协同作战能力。因此，对管理者而言，
努力培训团队成员，尤其是重视对能力差的员工的培训和补强，同时打造团队的合作精神，这是非常重要的管理课题。
```
16. 米格 —25效应
```
苏联研制生产的米格 -25喷气式战斗机，性能优越，广受世界各国青睐。然而，众多飞机制造专家发现：这家战斗机所使用的零部件与美国战机相比，要落后得多。那么为什么当
这些零部件组合起来，其整体作战性能达到甚至超过了美国最先进的战斗机呢？造成这种现象的原因是：米格公司在设计飞机时从整体考虑，对零部件进行了优化组合，使该机在
升降、速度、应急反应等方面超过了美国战机。这就是著名的“米格 -25效应”。
从米格 -25效应我们可以得出这样的结论：一个团队的内部结构是否合理，对其整体实力影响重大。当团队结构合理时，会产生“整体大于部分之和”的功效；反之，团队的整体功
能会小于各部分功能之和，甚至会出现负值。因此，重视团队建设不仅仅是优化各个团队成员的实力，更应该注重培养各成员之间的协作性，使大家默契地配合，高效地工作。
```
17. 华盛顿合作定律
```
一个人敷衍了事，两个人互相推诿，三个人则永无成事之日。多少有点类似于我们“三个和尚”的故事。人与人的合作不是人力的简单相加，而取决于团队成员之间的配合。
```
18. 苛希纳定律
```
在管理中，如果实际管理人员比最佳人数多2倍，工作时间就要多2倍，工作成本就多4倍；如果实际管理人员比最佳人数多3倍，工作时间就要多3倍，工作成本就多6倍。
在团队管理时，除了优化团队结构，还应首先优化团队人数，把团队人数控制在最适当的范围，才能保证团队的高效精干，同时又避免成本浪费。
```
19. 凝聚法则
```
凝聚法则说的是团队的凝聚力问题，凝聚力越大，团队就越有活力。一个有凝聚力的团队，对各个成员会形成一种吸引作用。个人的力量是有限的，如果能够得到集体成员的帮助
和支持，不仅有利于个人体会到集体的力量和魅力，更会在集体的帮助下取得更大的成就。
身为管理者，一定要让员工认识到团队的力量，要让每个成员都对团队负责，主动承担团队义务，形成团队意识，产生团队责任感、荣誉感，这样的团队才是有凝聚力和向心力的
，才能产生凝聚效应。
```
20. 懒蚂蚁效应
```
懒蚂蚁效应是日本北海道大学进化生物研究小组对三个分别由30只蚂蚁组成的黑蚁群的活动观察。结果发现。大部分蚂蚁都很勤快地寻找、搬运食物、少数蚂蚁却整日无所事事、
东张西望，人们把这少数蚂蚁叫做“懒蚂蚁”。 有趣的是，当生物学家在这些“懒蚂蚁”身上做上标记，并且断绝蚁群的食物来源时，那些平时工作很勤快的蚂蚁表现得一筹莫展，
而“懒蚂蚁”们则“挺身而出”，带领众蚂蚁向它们早已侦察到的新的食物源转移。原来“懒蚂蚁”们把大部分时间都花在了“侦察”和“研究”上了。它们能观察到组织的薄弱之处，同时
保持对新的食物的探索状态，从而保证群体不断得到新的食物来源。在蚁群和企业中，“懒蚂蚁”很重要，此现象被称为“懒蚂蚁效应”。
对于团队管理来说，管理者应该扮演好“懒蚂蚁”的角色，把精力放在战略制定、方向转移、观察市场、研究市场、把握市场和下属工作安排上来。要谨记：这些工作才是作为管理
者的你应该做的，而不要把时间耗费在鸡毛蒜皮的小事上。
```
21. 蚁群效应
```
蚁群效应是人们从蚂蚁群体的组织和分工中总结出来的灵活的组织建设和运转方式。蚂蚁有严格的组织分工，但它们的分工能够迅速根据环境作出调整。蚁群效应之所以成为 “高
效”的代名词，正是在于通过组织结构和岗位设置发挥了团队成员的组织能力。
蚂蚁有严格的组织分工，有明确的职责划分，做事很讲流程，这使得它们的工作充满效率。蚁群效应启示我们，在团队管理中，一定要明确各员工的职责，同时各个环节之间要配
合好。另外，团队工作流程中可能存在多余的环节，要想提高工作效率，最重要的办法就是发现和减掉那些多余的环节。这样不仅可以节约成本，更能提高效能。
```
22. 多米诺骨牌效应
```
多米诺骨牌（domino）是一种用木制或塑料制成的长方形骨牌。玩时将骨牌按一定间距排列成行，轻轻碰倒第一枚骨牌，其余的骨牌就会产生连锁反应，依次倒下。它阐释的是“一
荣难俱荣，一损易俱损”的连锁反应。
```

### 三、团队沟通法则

23. 坎特法则
```
尊重员工是人性化管理的必然要求，只有员工的私人身份受到了尊重，他们才会真正感到被重视，被激励，做事情才会真正发自内心，才愿意和经理打成一片，站到经理的立场，
主动与经理沟通想法探讨工作，完成经理交办的任务，甘心情愿为工作团队的荣誉付出。
```
24. 雷鲍夫法则
```
美国管理学家雷鲍夫从语言交往的角度，言简意赅地指出建立合作与信任的规律，那就是牢记这样8句话：
（1）我承认我犯过错误；
（2）你干了一件好事；
（3）你的看法如何；
（4）咱们一起干；
（5）不妨试试；
（6）谢谢您；
（7）咱们；
（8）您！
```
25. 霍桑效应
```
美国西部有一家电器公司叫霍桑工厂，为了提高工作效率，这个厂请来包括心理学家在内的各种专家，在约两年的时间内找工人谈话两万余人次，耐心听取工人对管理的意见和
抱怨，让他们尽情地宣泄出来。结果，霍桑厂的工作效率大大提高。这种奇妙的现象就被称作“霍桑效应”。
对团队管理而言，积极与员工沟通，多倾听它们的意见和想法，对他们给予关注，有利于激发员工的工作效率。
```
26. 杰亨利法则
```
杰亨利法则是指运用坦率真诚的沟通方式 。在企业里，人际的沟通是无可避免的，沟通问题也同样无可避免，开放、真诚、坦率是人际关系中的重要元素，是促进沟通渠道畅通的
有效保证。
```
27. 沟通的位差效应
```
来自领导层的信息只有20%-25%被下级知道并正确理解，而从下到上反馈的信息则不超过10%，平行交流的效率则可达到90%以上。
对企业管理者而言，要想上下级保持高效沟通，就有必要在整个企业内部建立一种平等沟通的机制。
```
28. 威尔德定理
```
威尔德定理认为有效的沟通始于倾听，人际沟通始于聆听，终于回答。由英国管理学家L·威尔德提出。
说的功夫有一半在听上。一问一答之间就可以受益无穷。在企业内部，倾听是管理者与员工沟通的基础。
```
29. 特里法则
```
“特里法则”讲的是美国田纳西银行前总经理特里指出的一句管理名言：承认错误是一个人最大的力量源泉，因为正视错误的人将得到错误以外的东西。
对管理者来说，由于自己的身份、职位特殊，犯错之后难以做到承认错误。也正因为如此，当管理者能够做到坦率承认错误时，他的形象往往瞬间变得高大起来，下属们会因此觉
得他更加可亲可敬。因此，在沟通中，如果管理者犯了错，千万别害怕承认。
```
30. 踢猫效应
```
踢猫效应是指对弱于自己或者等级低于自己的对象发泄不满情绪，而产生的连锁反应。“踢猫效应”，描绘的是一种典型的坏情绪的传染。人的不满情绪和糟糕心情，一般会沿着等
级和强弱组成的社会关系链条依次传递。由金字塔尖一直扩散到最底层，无处发泄的最弱小的那一个元素，则成为最终的受害者。其实，这是一种心理疾病的传染。
对企业管理者而言，对下属要好好沟通，而不要轻易发泄不满，因为人与人之间的泄愤会产生连锁反应。
```
31. 托利得定理
```
法国社会心理学家托利得提出，测验一个人的智力是否属于上乘，只看脑子里能否同时容纳两种相反的思想，而无碍于其处世行事。对管理者来说，脑子里能否同时容纳两种相反
的思想，既体现了领导者的胸怀和气度，又体现了领导者的智慧和仁厚。
```
32. 吉尔伯特法则
```
英国人力培训专家吉尔伯特提出，工作危机最确凿的信号，是没有人跟你说该怎样做。对管理者来说，当下属或员工积极与你谈论自己的想法，表达对公司发展的意见和建议，指
出公司存在的问题时，也是你的福气，是公司的福气，这也意味着他对公司的关心。
```
33. 摩斯科定理
```
打破沙锅问到底，这就是摩斯科定理的精髓。摩斯科定理主张刨根得根，问底知底。因为，当你询问他们问题时，他人的第一反应往往不假思考或者随意应付，只有继续追问下去
才能得到想要的答案。在营销中，摩斯科定理用处甚广，当然，也需要培养优秀的口才和足够的勇气。
```
34. 波克定理
```
美国庄臣公司总经理詹姆士·波克提出，只有在争辩中，才可能诞生最好的主意和最好的决定。
对企业管理者而言，在决策时，一定要想方设法发动团队成员集思广益，鼓励大家提出自己的观点和想法，即便大家的观点不一致，甚至截然不同，也有必要大胆地提出来，然后
在论辩中做出最好的决定。这就叫“真理越辩越明”。
```
35. 韦奇定理
```
美国洛杉矶加州大学经济学家伊渥·韦奇提出，即使你已有了主见，但如果有10个朋友看法和你相反，你就很难不动摇。
对于同一件事，大家各执一词、莫衷一是时，优秀的管理者往往会旗帜鲜明地提出自己的观点，做报晓的雄鸡，而没有主见的管理者往往人云亦云，服从大众的意见。
```

### 四、团队激励法则

36. 横山法则
```
日本社会学家横山宁夫提出自发的才是最有效的，激励员工自发地工作最有效并持续不断的控制不是强制，而是触发个人内在的自发控制，这种观点被称为横山法则。
有自觉性才有积极性，无自决权便无主动权。在管理的过程中，我们常常过多地强调了“约束”和“压制”，事实上这样的管理往往适得其反。如果人的积极性未能充分调动起来，规
矩越多，管理成本越高。聪明的企业家懂得在“尊重”和“激励”上下功夫，了解员工的需要，然后满足他。只有这样，才能激起员工对企业和自己工作的认同，激发起他们的自发控
制，从而变消极为积极。真正的管理，就是没有管理。
```
37. 信任法则
```
1968年的一天，美国心理学家罗森塔尔和L.雅各布森来到一所小学，说要进行7项实验。他们从一至六年级各选了3个班，对这18个班的学生进行了“未来发展趋势测验”。之后，罗
森塔尔以赞许的口吻将一份“最有发展前途者”的名单交给了校长和相关老师，并叮嘱他们务必要保密，以免影响实验的正确性。其实，罗森塔尔撒了一个“权威性谎言”，因为名单
上的学生是随便挑选出来的。8个月后，罗森塔尔和助手们对那18个班级的学生进行复试，结果奇迹出现了：凡是上了名单的学生，个个成绩有了较大的进步，且性格活泼开朗，
自信心强，求知欲旺盛，更乐于和别人打交道。这就是信任法则，亦称“罗森塔尔效应”，是一种社会心理效应，指的是教师对学生的殷切希望能戏剧性地收到预期效果的现象。
运用到管理中，就是要求管理者对下属要投入感情、希望和给予特别的引导，使他们充分发挥自身的主动性和创造性。
```
38. 六四法则
```
六四法则指的是对待员工，要给予六分表扬和四分批评。作为一名管理者，应履行好评价员工的职责，当下属表现好时应该表扬，当下属表现不好时应适当批评。为什么表扬与批
评要六四开呢？这是因为如果批评分量过大，表扬分量过小，可能会导致员工产生消极情绪；而表扬太多，批评太少，员工则容易产生骄气，有时甚至会产生误解，认为管理者有
意在给自己戴高帽，满足大家的虚荣心，久而久之会对表扬产生反感。
```
39. 彼得原理
```
彼得原理是美国学者劳伦斯·彼得在对组织中人员晋升的相关现象研究后得出的一个结论；在各种组织中，由于习惯于对在某个等级上称职的人员进行晋升提拔，因而雇员总是趋
向于被晋升到其不称职的地位。彼得原理有时也被称为“向上爬”理论。这种现象在现实生活中无处不在：一名称职的教授被提升为大学校长后无法胜任；一个优秀的运动员被提升
为主管体育的官员，导致无所作为。对一个组织而言，一旦相当部分人员被推到其不称职的级别，就会造成组织的人浮于事，效率低下，导致平庸者出人头地，发展停滞。将一名
职工晋升到一个无法很好发挥才能的岗位，不仅不是对本人的奖励，反而使其无法很好发挥才能，也给组织带来损失。
```
40. 总结法则
```
总结法则指的是当团队完成一项计划或工作时，管理者应该召开团队总结会，交流成功的经验，提出应该注意的问题，与员工共庆成功，相互激励。这样既可以鼓励员工把接下来
的工作做得更出色，又可以通过相互交流，进一步强化团队同甘共苦的一体感，将成功的喜悦转化为新工作的动力。
对个人的工作也可以采取这种激励法则，例如，当你委托下属负责某项工作，当他出色完成这项工作时，不妨请他单独谈一谈这次工作的感受，并对他提出表扬，激励他迎接下一
个挑战的信心。管理者这样做，可以让员工获得满足感、成就感和荣誉感，还可以增进彼此之间的信赖感。
```
41. 五分之三法则
```
五分之三法则说的是，让3个员工做5个员工的事，然后让他们领4个员工的薪水。这是一种薪酬激励的措施，它的原理在于：让3个员工做5个员工的事，可以节省2个用人成本；再
让这3个员工领4个员工的薪水，可以提升这3个员工的薪酬，让他们感受到激励。最终，企业节省了一名员工的用人成本，并激励了员工，可谓双赢。
```
42. 末位淘汰法则
```
GE公司前CEO杰克·韦尔奇提出，将员工划分为不同的类别，然后严格地加以区别对待，并根据员工的表现淘汰末位10%的员工，这就是末位淘汰法则，也叫活力曲线。
这个法则可以激发团队的活力，激励员工为避免淘汰而更加努力地工作。
```

### 五、团队用人法则

43. 古狄逊定理
```
古狄逊定理是由英国证券交易所前主管N·古狄逊提出的，点评管理是让别人干活的艺术。一个累坏了的管理者，是一个最差劲的管理者。
在现实生活中，我们会发现有不少管理者常常忙得焦头烂额，恨不得一天有48小时可用；或者常常觉得需要员工的帮忙，但是又怕他们做不好，以致最后事情都往自己身上揽。
虽然一个称职的管理者最好是一个“万事通”，但一个能力很强的人并不一定能管理好一家企业。管理的真谛不是要管理者自己来做事，而是要管理者管理别人做事。因此，聪明
的挂历者应该学会授权和分配工作，借助团队的力量去经营企业。
```
44. 倒金字塔管理法则
```
在大多数公司的管理中，采用的都是“正金字塔”管理模式，塔尖就是总经理或叫决策者，中间是中层管理者，最下面的是“First Line Staff”，即一线员工或叫执行者。然而，瑞
典的北欧航空公司总裁卡尔松却把这种管理模式颠倒了，这就是“倒金字塔管理法则”。
卡尔松的“倒金字塔管理法则”指的是把自己放在最下面，给自己命名为政策的监督者。他认为公司的总目标一旦制定下来之后，总经理的任务就是监督、执行政策，让员工达成目
标。最上面一层是一线工作人员，卡尔松称他们是现场决策者。这种管理策略让员工感觉到自己的重要性，并对自己分内负责的事情做出决定，有些决定甚至可以不必报告上司。
这极大地激发了员工的积极性。
```
45. 奥格尔维定律
```
奥格威法则来源于这样一个故事。美国奥格尔维·马瑟公司总裁奥格尔维召开了一次董事会，在会议桌上，每个与会的董事面前都摆了一个相同的玩具娃娃。董事们面面相觑，不知
何故。奥格尔维说："大家打开看看吧，那就是你们自己!"于是，他们一一把娃娃打开来看，结果出现的是：大娃娃里有个中娃娃，中娃娃里有个小娃娃。他们继续打开，里面的娃
娃一个比一个小。最后，当他们打开最里面的玩具娃娃时，看到了一张奥格尔维题了字的小纸条。纸条上写的是："如果你经常雇用比你弱小的人，将来我们就会变成矮人国，变成
一家侏儒公司。相反，如果你每次都雇用比你高大的人，日后我们必定成为一家巨人公司。"
对于管理者来说，应该任用比自己优秀的人，而不是任用比自己差的人。因为如果你只任用比自己差的人，那么他们就只能做出比你更差的事情。而如果你任用的都是比你更强的人
，则可以让公司成为巨人公司。
```
46. 不值得定律
```
不止的定律指的是，不值得做的事情，就不值得做好。它蕴含的道理再简单不过，但它的重要性却被人忽视。对管理者而言，在给员工安排工作时，务必牢记该定律的警示：如果
员工认为你交给他的工作时不值得做的事情，他往往会冷嘲热讽、敷衍了事，这样不仅成功率小，即便做成了这件事，他也不会有成就感。
对一个企业或团队来说，管理者要了解员工的性格特点，合理分配工作，比如，让员工负责有较强成就感、有一定的挑战和难度的工作，并在其完成工作时，给予及时的肯定和赞
扬，让员工感觉到自己所做的工作是值得的，这样才能激发职工的热情。
```
47. 特雷默定律
```
英国管理学家E·特雷默提出特雷默定律：每个人的才华虽然高低不同，但一定是各有长短，因此在选拔人才时要看重的是他的优点而不是缺点，利用个人特有的才能再委以相应责
任，使各安其职，这样才会使诸方矛盾趋于平衡。否则，职位与才华不能适合，使应有的能力发挥不出﹐彼此之间互不信服﹐势必造成冲突的加剧。在一个团队中，每个人各有所
长，但更重要的是领导者能将这些人依其专长运用到最适当的职位，使其能够发挥自己所长，进而让整个企业繁荣强盛。没有无用的人，只有不会用人的人。
```
48. 乔布斯法则
```
美国苹果电脑公司前老板史蒂夫·乔布斯说，他花了半辈子时间才充分意识到人才的价值。他曾说：“我过去常常认为一位出色的人才能顶2名平庸的员工，现在我认为能顶50名员
工。”这就是著名的乔布斯法则。
有了这种意识之后，乔布斯把大约四分之一的时间用来招募一流的人才。公司的高级管理者，也积极地向人才介绍公司的远景目标，以此留住人才。公司有一套缜密的人才招聘
流程，这样既可以让企业快速了解求职者，也可以让求职者以最快的速度了解公司，适应企业文化氛围和环境。
```
49. 大荣法则
```
号称日本两大百货公司之一的大荣百货公司创建于1957年。初创时的大荣公司只是大坂的一家小百货店，职工13人，后来扩展到经营糖果、饼干等食品和百货、大荣公司的经营决
策是：重视对人才的培养，由此走上了成功的道路。
企业的发达，就是人才的发达；人才的繁荣，就是事业的繁荣。因此，管理者应重视对员工培训和指导，帮助员工提高自身的素质和能力。
```
50. 酒井法则
```
日本企业管理顾问酒井正敬提出，企业在招人时使劲浑身解数，运用各种各样的方法，不如从自身做起，让公司变成一家优秀的企业，这样人才自然而然会主动找上门来。酒井法
则告诉我们，如果企业不加强留住人才的软、硬件假设，那么仅靠考聘是没有多大效果的，因为即便你今天找来了人才，明天人才也很容易流失。
所以，管理者要重视企业文化和制度的建设，努力将企业打造成充满魅力的大家庭，使大家不仅能赚到丰厚的薪酬，还能获得尊重，尽情展示才华，实现个人价值。这样，人才自
然云集。
```
