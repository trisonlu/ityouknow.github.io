# 书本摘要

## 管理哲学：一个人是工作，一群人是事业

1. “三个臭皮匠赛过诸葛亮”，团队协作永远大于孤军奋战
```
（1）优秀的团队，强调的是优势互补、相互协作
（2）让每个员工都融入团队中，彼此相互配合
（3）跨越“部门墙”，让企业各部门之间实现大团结
```
2. 团队成员不仅仅是你的下属，更是你的事业资本
```
purpose partners概念：把员工称为“同志”，即志同道合的伙伴
（1）激情万丈地向员工描述企业愿景
（2）对员工在工作中的付出表示认可
（3）尽可能多让员工参与到决策中来
```
3. 团队精神是企业最核心的竞争力
```
（1）营造“企业兴旺我光荣，企业衰亡我可耻”的文化氛围
（2）把“家人意识”融入团队建设中来
```
4. 凝聚力是团队的保障，创新力是团队的希望
```
（1）在实践与体验中寻找创新灵感
（2）以消费者的身份造访竞争对手
```
5. 好团队的标准：谁都可以用得上，谁都可以离得开
```
团队里的人谁都可以离开，包括管理者，这才不失为一个好的团队
（1）努力培养员工独立解决问题的能力
（2）不再过细地监督下属的工作，而要用好的结果要求下属
```
6. 打造协作性团队，实现每个人的最大利益
```
应该考虑的因素
（1）能力因素
（2）知识因素
（3）气质因素（气质指的是脾气、性格、秉性等）
（4）年龄因素
```
7. 形成利益和荣誉的共同体，才能形成牢固的团队
```
（1）明确地制定利益分配的方案
（2）在荣誉激励中加入感情的“戏份”
```

## 第一招 自我管理：用人格感召团队，用魅力影响团队

1. 不断丰富自己的知识结构，做学者型管理者
```
非学无以广才
管理者有三大能力：技术能力、人际能力和分析能力，支撑这些能力的事学习能力
（1）向书本学习
（2）向实践学习
（3）向优秀者学习
```
2. 做情绪的主人，管理者应有情绪掌控力
```
（1）给自己敲警钟，提醒不良情绪的危害
（2）运用自我观察法控制不良情绪
    ①深呼吸，让自己集中注意力
    ②观察不良情绪状态下你是想逃避还是想对抗
```
3. 修炼非权力性领导力：让员工自觉追随你
```
（1）自信、谦虚、胸怀宽广
（2）吃苦耐劳的低调精神
（3）平易近人的待人态度
```
4. 自我要求：用放大镜管人，用望远镜管事
```
（1）把“用放大镜管人”和“用望远镜管事”作为对自己的要求
（2）严格要求团队成员保持注重细节的工作态度
```
5. 不要和下属争功劳，争了功劳失了心
```
（1）功劳确实是下属的，毫无疑问要给下属
（2）即便是你的功劳，也别忘了与下属分享
（3）下属尽力了，但成绩不突出，依然要记上一功
```
6. 无关紧要的事情，妥协一下更能彰显你的气度
```
（1）有些事可以先缓一缓，不急于一时解决
（2）有些事无须挑明，装一装糊涂给员工留面子
```
7. 必要时做员工的挡箭牌，保护员工得人心
```
（1）管理者决策失误，导致下属工作出差错时，应主动揽过
（2）管理者督察不力，导致下属出现差错时，应主动揽过
（3）管理者用人不当，导致下属工作出错，应主动揽过
```

## 第二招 制度管理：制度才是真正的老板

1. 严格的制度是企业健康成长的保险栓
```
（1）完善
（2）公平
（3）服众
```
2. 按制度办事，没有规矩不成方圆
```
（1）制定具体的可操作的企业管理制度
（2）制度不仅有规定，还应有处置的方式
```
3. 合理的制度设置应该兼顾公平和效率
```
（1）制度的制定初衷要考虑清楚
（2）制度的设置应该兼顾公平和效率
（3）一个制度的出台应该以保护积极实干者为出发点
```
4. 制度就是原则，制度面前人人平等
```
（1）管理者应该积极做表率
（2）按制度办，树立制度的威信
```
5. 执行力是制度权威存活的生命之源
```
（1）处罚不是目的，而是手段
（2）有罚也要有奖，奖罚分明
```
6. 你不讲制度，大家就会跟你讲条件
```
（1）没有制度的公司，一定要赶快建立和完善制度
（2）有制度一定要严格执行制度，而不是法外开恩
```
7. 杜绝人情，它是制度的腐蚀剂
```
（1）区别人情化与人性化
（2）杜绝人情，但要讲人性
```
8. 杜绝“下不为例”的借口，制度面前没有任何理由
```
（1）制度一旦制定，就要严格执行
（2）按制度办事别忘了教育员工
```

##  第三招 授权管理：管的少，团队才能管得好

1. 合理授权才能凝聚起团队的力量
```
（1）授权要适度，别让下属压力太大，以免影响执行效率
（2）授权之后切忌胡乱干涉，以免打击下属的自信心和积极性
```
2. 授权管理的原则：分散权力，总揽大局
```
（1）逐级授权原则
（2）权责明确原则
（3）抓总线、放支线原则
```
3. 管理者不做太多决策，只做重大决策
```
（1）常问自己：这件事我有必要做吗？
（2）明确自己的角色，有所为，有所不为
（3）大事与小事，其实是相对而言的
```
4. 授权的前提是找到合适的人选
```
（1）看下属是否具有完成某项工作的特长和能力
（2）充分考虑授权对象的个性、脾气、气质等
```
5. 有效授权和盲目放权不是“同门师兄”
```
怎样过问和监督才能避免给下属造成不信任感
（1）讲明工作要求及完成期限
（2）适当关注，而不是质疑
```
6. 最有效的授权是让员工各尽其责
```
（1）人才适合干什么，就让他干什么
（2）划分职权，分配工作，让员工清楚自己的职责
```
7. 缺乏信任的授权不可能达到授权的目的
```
（1）在下属犯错之后，依然信任他、授权给他
（2）授权给那些不太出众的下属，激起他们的信心
```
